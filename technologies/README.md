# EcoWaveProjectManagement — Documentación técnica

Guías introductorias del stack tecnológico del proyecto, escritas en español para desarrolladoras backend que se incorporan al equipo.

---

## Contenido

### [Frontend](frontend/README.md)
Stack React + Vite + TypeScript. 30 tecnologías documentadas con analogías del mundo C#/.NET.

Incluye: React, Vite, Tailwind CSS, shadcn/ui, Zustand, TanStack Query, TanStack Router, ESLint, Vitest y más.

### [Backend](backend/README.md)
Stack ASP.NET Core + Dapper + Clean Architecture. 16 tecnologías y patrones documentados.

Incluye: ASP.NET Core Web API, Dapper, Clean Architecture, módulos funcionales, xUnit, NSubstitute, Shouldly y más.

### [Lista de tecnologías](TECHNOLOGY_LIST.md)
Inventario completo de todas las tecnologías del proyecto agrupadas por categoría (frontend, backend, bases de datos, DevOps).

### [Integración](integracion/README.md)
Cómo se comunican el frontend y el backend: proxy Vite, CORS, Docker Compose, flujo end-to-end de una petición y guía para añadir un endpoint nuevo.

---

## Por dónde empezar

Si eres nueva en el proyecto, lee en este orden:

1. **[Backend](backend/README.md)** — empieza por lo que ya conoces
2. **[Integración](integracion/README.md)** — entiende cómo frontend y backend se conectan
3. **[Frontend](frontend/README.md)** — explora el stack React cuando necesites tocar la UI
